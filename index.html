Mapa de estudos

Roadmap semestral

O que estudar em cada semestre, com prioridades e dicas práticas alinhadas à grade oficial.

Módulo I 1º Semestre Introdução
5 disciplinas teóricas + 455h total
🧠
Lógica de Programação
A disciplina mais importante do 1º semestre. Aqui você aprende variáveis, condicionais, laços e funções — a base de tudo. Dedique pelo menos 2h extras por semana.
CORE
💻
Ambientação em Tecnologia da Informação
Panorama da área: hardware, software, sistemas operacionais, carreira em TI. Ótima para entender onde o curso te leva.
INTRO
🖥️
Fundamentos de Sistemas de Informação
O que é um sistema de informação, tipos (ERP, CRM, BI) e papel no negócio. Contexto para tudo que vem depois.
INTRO
⚖️
Ética e Direitos Humanos / Fundamentos de Administração
Formação humanística obrigatória: LGPD, direitos digitais, gestão de projetos e pessoas.
FORMAÇÃO
💡 Prioridade máxima: Lógica de Programação. Se travar aqui, revise algoritmos com pseudocódigo e fluxogramas antes de qualquer linguagem real.
Módulo II 2º Semestre Sistemas e BD
5 disciplinas teóricas + 425h total
🔷
Programação Orientada a Objetos
Os 4 pilares: encapsulamento, herança, polimorfismo e abstração. Geralmente em Java ou Python. É o maior salto conceitual do curso.
POO
📊
Algoritmos e Estruturas de Dados
Listas, pilhas, filas, árvores, grafos e algoritmos de ordenação/busca. Essencial para entrevistas técnicas.
PROG
🗃️
Sistemas de Banco de Dados
Modelo relacional, SQL básico (SELECT, INSERT, UPDATE, DELETE), normalização e modelagem ER. Toda empresa usa banco de dados.
BD
⚙️
Engenharia de Software I
Ciclo de vida do software, requisitos, UML e documentação. Aprende a planejar antes de codar.
CORE
💡 Neste semestre você escreve seus primeiros programas reais. Pratique POO criando projetos pequenos: um sistema de cadastro, uma calculadora OO.
Módulo III 3º Semestre Web
5 disciplinas teóricas + 425h total
🌐
Desenvolvimento Web
HTML, CSS, JavaScript, HTTP, APIs REST e framework web (PHP/Laravel, Node.js ou similar). A disciplina mais prática do semestre.
WEB
🗄️
Banco de Dados Avançado
Otimização de queries, índices, stored procedures, transações, triggers e introdução a NoSQL (MongoDB).
BD
🔢
Matemática Discreta
Conjuntos, lógica proposicional, grafos, combinatória — base teórica de algoritmos e criptografia.
TEORIA
🌍
Redes de Computadores
TCP/IP, modelo OSI, DNS, HTTP/HTTPS, protocolos e segurança básica de rede. Fundamental para qualquer dev web.
INFRA
💡 Momento de montar o portfólio: construa um projeto web completo (front + back + banco) para mostrar aos recrutadores.
Módulo IV 4º Semestre Avançado
4 disciplinas teóricas + 390h total
📱
Programação para Dispositivos Móveis
Android (Kotlin/Java) ou React Native/Flutter. App mobile no portfólio é um diferencial enorme no mercado.
MOBILE
🔐
Segurança de Sistemas
OWASP Top 10, autenticação, criptografia, SQL injection, XSS, HTTPS e LGPD aplicada. Todo dev precisa saber isso.
SEGURANÇA
🧪
Qualidade de Software
Testes unitários, integração, QA, TDD, cobertura de código e revisão de código (code review).
QA
🏗️
Arquitetura de Software
Padrões de design (MVC, SOLID, Design Patterns), microsserviços, monolito vs. distribuído e clean architecture.
ARQ
💡 Aqui você já pode estar estagiando. Leve os projetos de mobile e segurança para o portfólio — são os que mais impressionam recrutadores.
Módulo V 5º Semestre Especialização
4 disciplinas teóricas + 325h total
⚙️
Sistemas Operacionais
Processos, threads, gerência de memória, sistemas de arquivos, Linux — base para DevOps e cloud.
INFRA
🔄
Metodologias Ágeis
Scrum, Kanban, XP, sprints, backlog, daily e retrospectiva. O mercado usa Scrum em 80% dos projetos de software.
ÁGIL
📈
Big Data e Análise de Dados
Hadoop, Spark, Python para dados (Pandas, NumPy), visualização (Power BI / Tableau) e SQL analítico.
DADOS
☁️
Computação em Nuvem
AWS, Azure ou GCP: IaaS, PaaS, SaaS, containers (Docker), orquestração (Kubernetes) e CI/CD.
CLOUD
💡 Último semestre: finalize o portfólio, obtenha uma certificação (AWS Cloud Practitioner, Google Associate) e prepare o LinkedIn.
